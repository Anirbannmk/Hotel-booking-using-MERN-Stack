…or create a new repository on the command line
echo "# MERN-Hotelbooking-website" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/Anirbannmk/MERN-Hotelbooking-website.git

…or push an existing repository from the command line
git remote add origin https://github.com/Anirbannmk/MERN-Hotelbooking-website.git
git branch -M main
git push -u origin main

To update your project on GitHub after deleting files from your local project, you’ll need to follow


git status
git add -A
git commit -m "Removed unnecessary files and updated the project"
git push origin main
